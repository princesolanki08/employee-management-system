# employee-management-system

## 📌 Project Overview

The Employee Management System is a role-based application designed to streamline employee data management and organizational workflows. It provides dedicated access for both administrators and employees, ensuring secure and efficient handling of operations.

---

## 🚀 Key Features

### 🔐 Role-Based Access
- Admin and Employee login system
- Secure authentication and authorization

### 👨‍💼 Admin Functionalities
- Add and manage employee records
- View complete employee details
- Approve or reject leave requests
- Monitor overall employee activities

### 👤 Employee Functionalities
- Signup and login functionality
- View personal profile and details
- Apply for leave requests
- Track leave status (approved/rejected)

### 📧 Automated Notifications
- Sends email notifications to employees on special occasions (e.g., birthdays)

---

## 🎯 Project Highlights

- Implements real-world business logic
- Demonstrates role-based system design
- Covers CRUD operations with additional workflow features
- Integrates database and email functionality
